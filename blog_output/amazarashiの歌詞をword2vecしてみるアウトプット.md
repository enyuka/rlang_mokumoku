## で、結果どうなのか？

なんかいまいちイメージと違う（厄介なオタク）

原因としては、訓練したときのベクターが少なすぎた可能性がある。
`model_amz = train_word2vec("/Users/nishiguchi/dev/rlang_mokumoku/rmecab_text_amz_list.bin", vectors = 30, window = 4, threads = 2)`

あと、訓練したときのデータでノイズが混じっていて、
カバー曲が入っているので除外しました。

### ベクターを上げて再挑戦

`model_amz = train_word2vec("/Users/nishiguchi/dev/rlang_mokumoku/rmecab_text_amz_list.bin", vectors = 1000, window = 4, threads = 2)`

一気に1000にしました。最初100にしてみましたが、そんなに遅くなかったので1000まで上げました。
ただ、1000でもそんなに遅い感じはしない。
同じ単語をチョイスして聞いてみる。

先に結論から書くと、1000にすると近い単語が変わり、
グッと印象が近づいた気がする。
